Unique Features The internal memory allows the interface to be used standalone without any mobile or tablet. For example, for long-term logging of parameters and fault codes or for a quick diagnostic scan. Fault codes and other notifications can be displayed with both blinking LEDs and sound.

Infocar Diagnostics is a free app that can be downloaded from Google Play. It enables you to check vehicle health, read and erase fault codes, view realtime parameters, log data and much more.
